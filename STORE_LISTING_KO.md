# Chrome Web Store Listing (Korean)

**[Taillog 설치하기](https://chromewebstore.google.com/detail/taillog-console-network-l/nomgkaaaojblgcgkmiepaaggbfpeeobm)**

## 짧은 설명

```text
콘솔 로그와 네트워크 요청을 한 화면에서 보는 프론트엔드 디버깅 패널.
```

## 짧은 설명 대안

```text
전체 DevTools를 열지 않고 로그, 요청, 클립보드, React 컴포넌트를 빠르게 확인하세요.
```

```text
콘솔, 네트워크, 보관함, 로케이터를 담은 깔끔한 프론트엔드 디버깅 익스텐션.
```

## 상세 설명

```text
Taillog는 프론트엔드 개발자를 위한 Chrome 디버깅 익스텐션입니다.

가장 자주 보는 신호를 먼저 보여주고, 필요할 때만 더 넓은 창으로 확장해서 볼 수 있도록 설계했습니다.

핵심 기능
- Console 로그
- Network 요청

복잡한 DevTools 흐름 대신, 팝업에서 빠르게 훑고 필요한 항목만 자세히 확인할 수 있습니다.

Taillog를 쓰는 이유
- 첫 화면이 단순해서 바로 상황 파악이 쉽습니다
- 최근 로그와 요청을 빠르게 훑을 수 있습니다
- Payload와 Response를 인라인으로 확인할 수 있습니다
- 보관함과 React 로케이터를 보조 도구로 함께 사용할 수 있습니다
- 영어와 한국어 UI를 지원하고 설정이 유지됩니다
- 모든 데이터는 로컬에만 저장됩니다

주요 기능

Console
- log, info, warn, error를 실시간으로 확인
- 단순한 레벨 필터 제공
- 필요한 항목만 펼쳐서 전체 payload 확인

Network
- fetch와 XMLHttpRequest를 실시간으로 확인
- 메서드 필터와 URL/경로 검색 지원
- Payload와 Response 상세를 인라인으로 확인
- 요청/응답 데이터를 빠르게 복사 가능

Stash
- 복사한 텍스트를 자동으로 저장
- 자주 쓰는 텍스트를 빠르게 다시 복사
- 디버깅 중 필요한 문자열을 가까이에 유지

Locator
- Alt 또는 Option으로 React 컴포넌트 탐색
- 메타데이터가 있으면 선호 IDE에서 소스 열기
- 선호 IDE와 활성 상태를 로컬에 저장

추가 편의 기능
- 마지막으로 열었던 탭을 기억
- 영어/한국어 UI 지원
- 더 넓게 볼 수 있는 별도 창 모드 제공

개인정보 및 데이터
- 모든 데이터는 내 기기에만 저장됩니다
- 계정이 필요하지 않습니다
- 외부 서버로 동기화하지 않습니다

가볍고 빠른 프론트엔드 디버깅 패널이 필요하다면 Taillog가 그 흐름에 잘 맞습니다.
```

## 스토어 포지셔닝 메모

- 핵심 가치는 `Console + Network`를 먼저 보여주는 것입니다.
- `Stash`, `Locator`는 보조 도구로 소개합니다.
- 다국어 지원과 별도 창 모드는 완성도를 보여주는 포인트로 활용합니다.
- 기능 나열보다 실제 사용 흐름 중심 문구를 우선합니다.

## 추천 메타데이터

- 키워드: `console logger`, `network logger`, `frontend debugging`, `react debugging`, `chrome extension`
