### Short Description (under 132 characters)
```
Console logs, network requests, and clipboard history in one place. A lightweight debugging tool for web developers.
```

### Detailed Description
```
Taillog is an all-in-one debugging tool for web developers.

Key Features

Console Tab
• Monitor console.log, error, warn, and info messages in real-time
• Filter by log type to quickly find what you need
• Auto-formatted JSON data for better readability

Network Tab
• Real-time monitoring of fetch and XMLHttpRequest calls
• Filter by GET, POST, PUT, DELETE methods
• Search by URL to quickly find specific requests
• Copy request Payload and Response with one click

Stash Tab
• Automatically saves copied text
• View your recent clipboard history at a glance
• Re-copy to clipboard with a single click

Why Taillog?

• Lightweight & Fast: Works without impacting browser performance
• Privacy First: All data stored locally only, no external transmission
• Simple UI: Clean interface with just the features you need

Debug quickly without opening DevTools!
```
